## 테스트
### 테스트 함수 만들기
```
control + shift + T
command + shift + T
```

### 테스트 함수 실행
~~~
control + shift + r
~~~

## 함수
### 리턴 값 자동 생성
```
control + alt + V
```

### 함수 추출하기
```
option + command + m
```

### 함수 파라미터 추출하기
```
option + command + p
```

## 코드
### 줄 삭제
```

```

### 줄 복사
```
control + D
```
