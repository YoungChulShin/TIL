## LiveTemplate 적용

적용 방법
1. 메뉴 이동
   - Preference -> Editor -> Live Templates 이동
2. Add -> Live Template 
3. 원하는 정보 입력
   - Abbreviation: 단축키 값
   - Description: 설명
   - Template text: 생성되는 코드
   - declaration: 선언되는 곳 설정. 자바면 자바를 선택

기타 설정
- 변수 설정: `$이름$`으로 선언
- Edit variables에서 변수에 대한 기본 값 등 설정 가능
