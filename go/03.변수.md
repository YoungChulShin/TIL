# 변수
## var
변수 선언
```go
var a int   // 기본 값 0
var b string    // 기본 값 null
var c, d, e int // 여러개 동시 선언
var f, g, h int = 1, 2, 3   // 동시 선언 및 할당
var i float32 = 12.3
var k = 12.3    // 타입 선언 없이 할당
var l = "hello go"
var m bool = true
var n = false
```

여러 변수를 동시에 선언
```go
var (
    name      string = "machine"
    height    int32
    weight    float32
    isRunning bool
)
```

짧은 변수 선언
```go
// 변수 선언
shortVar1 := 3
shortVar2 := "Test"
shortVar3 := false

// if 문에서 사용
if i := 10; i < 9 {
    fmt.Println("성공: ", i)
} else {
    fmt.Println("실패: ", i)
}
```

## const
상수 선언
```go
const a String = "hello"
const b = "hello"
const c, d = 10, 12.9
const (
    name = "hello"
    age = 10
)
```
