# 실행
## run, build, install
run
- 명령어: `go run <<go file>>`
- 코드를 바로 실행

build
- 명령어: `go build <<go file>>`
- 실행가능한 파일을 생성. 컴파일

install
- 명령어: `go install`
- 의존관계를 포함하는 최종적인 실행가능한 파일로 생성
- 소스 파일이 있는 상위 폴더명으로 생성


## gofmt 
코드 스타일을 통일되게 변경해주는 기능
- `gofmt -h`: 사용법 확인
- `gofmt -w <<file or folder>>`: 적용
